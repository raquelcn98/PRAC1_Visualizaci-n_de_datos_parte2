# Proyecto de Visualización – Trastornos neurológicos (GBD 1980–2023)

Visualización interactiva en HTML publicada con GitHub Pages.

Este proyecto presenta una visualización interactiva desarrollada en R y Plotly que analiza la carga global de los trastornos neurológicos entre 1980 y 2023 a partir de datos del estudio Global Burden of Disease (GBD).
La herramienta permite explorar la evolución temporal, comparar hombres y mujeres, identificar desigualdades regionales y analizar qué enfermedades generan mayor mortalidad o pérdida de años de vida saludable (AVISA).
El objetivo es facilitar la comprensión de un problema de salud pública complejo mediante visualización interactiva orientada tanto a público general como especializado.
